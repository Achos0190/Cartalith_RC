Cartalith — Worldbuilding System

Release Candidate v0.01 · created by V. Post

Cartalith unifies three purpose-built, zero-dependency tools into one cohesive, offline, single-file worldbuilding system: a procedural world engine, a cartographic editor, and an asset-pack compiler. Each tool runs intact; a generated world flows straight into the cartographer, authored art flows into the engine, and a single project file saves everything.

Generate

Elevation Foundation v0.144 — procedural tectonics, climate, hydrology, erosion, biomes, LOD atlas.

Cartograph

Cartalith V1.915 — routes, settlements, painted biome/terrain grids, politics timeline, journey planner.

Asset Packs

Asset Pack Compiler — author texture/icon/symbol packs (schema 2) with a sprite-sheet slicer.

Each tool's own Save / Load / Import buttons remain fully functional inside its pane. Project ▾ → Save additionally bundles all three tools' native saves into one coherent .cartalith file, and Open restores them — nothing from any tool is lost.
